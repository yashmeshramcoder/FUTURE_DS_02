# SaaS Customer Churn Analysis

## Overview
This project presents an analysis of SaaS customer churn data using Microsoft Excel.  
The goal of the analysis is to understand customer behavior, identify churn patterns, and generate meaningful business insights.

## Project Objectives
- Analyze customer churn trends
- Understand factors affecting customer retention
- Identify patterns in customer accounts and subscriptions
- Create a structured business analysis report

## Tools & Technologies Used
- Microsoft Excel
- MS Word

## Dataset Details
The dataset contains customer-related information including:
- Customer accounts
- Subscription details
- Churn status
- Revenue information
- Customer activity data

## Files Included
- `ravenstack_accounts_done.csv.xlsx` – Dataset used for analysis
- `SaaS_Churn_Analysis_Report.pdf` – Final project report

## Key Insights
- Churn patterns were analyzed using Excel
- Customer retention trends were identified
- Business insights and observations were documented in the report

## Conclusion
This project demonstrates how customer data can be analyzed to better understand churn behavior and support business decision-making.
